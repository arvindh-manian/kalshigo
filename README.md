# KalshiGo

KalshiGo is a Go package providing low-level bindings to the Kalshi API. Inspired by [DiscordGo](https://github.com/bwmarrin/discordgo/tree/master).

## Status
| Section      | Support Status |
| ------------ | -------------- |
| market       | ✅             |
| exchange     | ❌             |
| auth         | ❌             |
| collection   | ❌             |
| portfolio    | ❌             |
| sockets      | ❌             |
